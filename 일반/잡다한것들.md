# 올림 표기법

```mm
def ceil(num):
  rem = 1 if num % 1 != 0 else 0
  return int(num // 1 + rem)
```

소수점 있는지 확인하는 ceil 함수 정의한 것.

## 다른 올림 표기법

i = 95
j = 2
식 Cal = (100 - i / 2) = 2.5
식 Cal = (100 - i // 2) = 2
5 / 2 = 2.5
5 // 2 = 2

그러나
i = 95
j = 2
식 cal = (i-100 / 2) = -2.5
식 cal = (i-100 // 2) = -3

-5 / 2 = -2.5
-5 // 2 = -3

음수의 경우 자동으로 -5 // 2를 해도 자동으로 내림 처림 되어서 -3이 되므로

-(i - 100 // j) = 3이라는 올림과 같은 결과를 가져올 수 있다.
